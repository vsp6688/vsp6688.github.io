生成／复制 PS1
PS1是UNIX、MacOS、Linux的系统变量，用来定义终端窗口中命令输入提示符前的显示内容。
本页面通过添加、删除内容、更改背景、前景颜色、拖拽排序，来实时预览终端显示效果。
点击生成按钮，自动生成PS1并复制到剪贴板中，将复制内入写入系统配置文件，就可以改变终端显示样式。
尽量不要使用中文及非键盘字符作为自定义内容，可能会引起输入位置错位。
联系制作者 微博:王若小智 GitHub
